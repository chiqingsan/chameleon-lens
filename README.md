# Chameleon Lens

Chameleon Lens 是一个用于 MECCHA CHAMELEON 的外部覆盖层工具，提供 ESP、雷达、外观调整和快捷键控制。建议在窗口化或无边框窗口模式下使用。

> 本项目仅用于学习、研究和界面实验。使用第三方工具可能违反游戏服务条款，请自行承担风险。

## 功能

- **ESP 显示**：显示目标圆点、名称、距离、射线、本地标记和边缘提示。
- **雷达面板**：显示目标相对位置，可调整范围、尺寸、位置和亮度。
- **外观调整**：修改菜单透明度、圆点半径和目标颜色。
- **快捷键**：配置菜单显隐、覆盖层总开关、ESP 绘制和雷达面板开关。
- **调试信息**：查看连接状态和最近错误，并打开日志目录。
- **自动保存**：菜单设置会自动保存，下次启动继续使用。

## 使用要求

- Windows 10/11
- MECCHA CHAMELEON 使用窗口化或无边框窗口模式
- 如果拿到的是源码包，需要安装 Python 3.11 或更高版本

## 启动

如果使用打包好的版本，直接运行 `.exe` 文件。

如果使用源码包，双击运行：

```bat
run.bat
```

首次启动会自动准备运行环境，可能需要等待一会。之后再次启动会更快。

## 基本用法

1. 启动 Chameleon Lens。
2. 启动 MECCHA CHAMELEON。
3. 等待标题栏状态变为“已连接 · 游戏进程”。
4. 默认按 `F1` 显示或隐藏菜单。
5. 在菜单中调整 ESP、雷达、外观和快捷键。

如果游戏尚未启动，菜单会显示“未连接 · 等待进程”。此时程序会保持运行，并自动等待游戏进程出现。

## 菜单页面

### ESP

控制屏幕 ESP 的显示内容，包括目标圆点、名称、距离、射线、本地标记和边缘提示。

关闭“ESP 绘制”只会隐藏屏幕 ESP，不会影响雷达面板。

### 雷达

控制覆盖层中的雷达面板。可以调整雷达范围、尺寸、显示位置和盘面亮度。

### 外观

调整菜单透明度、圆点半径和目标颜色。颜色修改后会立即影响预览和覆盖层显示。

### 快捷键

配置常用开关的快捷键。菜单显隐默认是 `F1`，其他快捷键默认未设置。

### 调试

查看当前连接状态、最近错误和日志目录。遇到连接或显示问题时，可以先从这里打开日志目录。

## 配置和日志

使用源码包运行时：

- 配置文件：`config.json`
- 日志目录：`logs/`

使用打包版本运行时：

- 配置和日志保存到 `%LOCALAPPDATA%\Chameleon Lens`

## 常见问题

### 菜单一直显示“未连接 · 等待进程”

确认游戏已经启动。如果游戏以管理员权限运行，Chameleon Lens 也需要使用相同权限启动。

### 覆盖层位置不对或看不到

确认游戏是否为窗口化或无边框窗口模式。独占全屏可能导致覆盖层无法正常显示。

如果先打开 Chameleon Lens、再打开游戏，工具会自动重新置顶覆盖层并在游戏画面初始化后重建连接。若屏幕 ESP 仍不可见，先确认 ESP 页里的“ESP 绘制”是否开启，或检查是否被快捷键关闭后保存。

### 游戏更新后显示异常

游戏更新可能导致覆盖层无法正确识别目标，需要等待适配。
